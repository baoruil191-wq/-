#  AI 自动化营销文案工作流（Workflow）上手指南

欢迎来到舞肌科技！本工作流将帮助你一键将“商品关键词”自动化转化为“高点击率的小红书爆款文案”。整个流水线包含 3 个核心节点，顺着本指南，小白用户也能在 **3分钟内** 跑通第一次自动化。

---

##  核心工作流节点与配置说明

| 节点顺序 | 节点名称 |  这一步你要干嘛？（核心动作） |  防呆避坑提示 |
| :--- | :--- | :--- | :--- |
| **01** | `Data_Input` (数据输入) | 输入你想要推广的商品核心关键词（如：生椰拿铁、滑雪板）。 | 关键词之间用英文逗号隔开，数量建议 **3-5个** 效果最佳。 |
| **02** | `LLM_Generation` (大模型生成) | 调教 AI 核心大脑。需要配置大模型的 Prompt（提示词）与核心参数。 | **高危流失点**：必须使用 Few-Shot（少样本）约束，严禁空对空提问！ |
| **03** | `Format_Output` (格式化输出) | 系统会自动将大模型生成的文案进行清洗，并 100% 自动化入库。 | 本节点由系统后台自动承接，**运营无需任何代码操作**。 |

---

##  核心参数调优（针对节点 02）

在配置大模型节点时，有两个北极星参数直接决定文案质量：

### 1. Temperature（情绪控制阀）
* **写严肃技术文档**：推荐设置为 `0.3`（AI 会非常老实、严谨）。
* **写营销小红书文案**：推荐设置为 `1.0 - 1.2`（释放 AI 的创造力，文案更抓人眼球）。

### 2. Prompt 的 Few-Shot（少样本）标准示范
为了彻底消灭系统报错，请严格将以下 JSON Schema 结构复制并粘贴到你的提示词框内：

```json
{
  "brand": "你的品牌名称",
  "keywords": ["关键词1", "关键词2"],
  "output_format": "Markdown"
}
